Classpublic class OSCConnectionEvent
InheritanceOSCConnectionEvent flash.events.Event

An OSCConnection dispatches an OSCConnectionEvent object when an osc packet is recieved or send. Ther are two types of data event:
  • OSCConnectionEvent.OSC_PACKET_IN: dispatched for osc data received.
  • OSCConnectionEvent.OSC_PACKET_OUT: dispatched for osc data send.
